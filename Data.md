# Dataset Setup for PixFoundation

## PixMMVP
* Referring expressions and segmentation annotations are available under data/pixmmvp/
* Setup the dataset directory to the following

```
|--- MMVP
   |--- MMVP Images
   |--- Questions.csv
   |--- Objects.csv
   |--- Segmentations.json
   |--- visual_patterns.csv
```
